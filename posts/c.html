C Programming Language

C is a powerful, general-purpose programming language that allows you to write efficient programs. It provides low-level access to memory and system resources, making it ideal for system programming like operating systems. C's syntax and structure are foundational, influencing many modern programming languages like C++ and Java.

Starting Code

This is a simple C starting code:

#include <stdio.h>

int main(void) {
  printf("Ahmed notes\n");
  return 0;
}

Note: "void" in int main(void) is optional, which means the code can be written as:

#include <stdio.h>

int main() {
  printf("Ahmed notes\n");
  return 0;
}

Comments

Comments in C can be written in two ways:

1. Single-line comment:

// This is a comment
int main() {
  printf("Ahmed notes\n");
  return 0;
}

2. Multi-line comment:

/* This is a
multi-line 
comment */
int main() {
  printf("Ahmed notes\n");
  return 0;
}

Escape Sequences

Escape sequences in C are special sequences used to represent certain special characters within strings:

  • \n: Newline
  • \t: Horizontal Tab
  • \\: Backslash
  • \": Double Quote
  • \': Single Quote
int main() {
  printf("Ahmed\nnotes\nare\ndecent");
  return 0;
}

Variables

In C, a variable is a named storage location in memory used to hold a value:

1. int (Integer)

int age = 25;

2. float (Floating Point)

float temperature = 36.5;

3. double (Double Precision)

double pi = 3.14159;

4. char (Character)

char grade = 'A';

5. _Bool (Boolean)

_Bool isTrue = 1;

Displaying Variables

In C, you can display the values of variables using printf:

1. int (Integer)

int age = 25;
printf("Age: %d\n", age);

2. float (Floating Point)

float temperature = 36.5;
printf("Temperature: %.2f\n", temperature);

3. double (Double Precision)

double pi = 3.14159;
printf("Value of Pi: %.5lf\n", pi);

4. char (Character)

char grade = 'A';
printf("Grade: %c\n", grade);

5. _Bool (Boolean)

_Bool isTrue = 1;
printf("Boolean Value: %d\n", isTrue);
